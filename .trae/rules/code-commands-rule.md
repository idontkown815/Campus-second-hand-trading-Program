# 项目技术栈定义

## 后端技术
- **框架**：Django
- **语言**：Python 3.12.3

## 数据库
- **主数据库**：MySQL 8.0
- **ORM**：Django ORM

## 前端技术
- **UI框架**：Bootstrap 5
- **JavaScript库**：jQuery

## 存储服务
- **图片存储**：阿里云 OSS

## 技术栈使用原则
- 严格遵循Django最佳实践
- 使用Django ORM进行数据库操作，避免直接编写SQL
- 前端使用Bootstrap 5构建响应式界面
- 图片文件统一存储在阿里云OSS，本地仅存储文件路径
